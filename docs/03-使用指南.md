# 03-使用指南

本文档提供 Galay-Kernel 的详细使用说明、最佳实践和常见场景。

## 核心概念

### Runtime vs Scheduler

| 组件 | 作用 | 使用场景 |
|------|------|----------|
| Runtime | 统一管理多个调度器 | 应用程序入口，自动管理资源 |
| IOScheduler | IO 事件调度 | 网络 IO、文件 IO |
| ComputeScheduler | CPU 任务调度 | 计算密集型任务 |
| TimerScheduler | 定时器调度 | 定时任务、超时控制 |

### 协程生命周期

```cpp
// 1. 创建协程
Coroutine coro = myCoroutineFunction();

// 2. 提交到调度器
scheduler->spawn(std::move(coro));

// 3. 协程执行
// - co_await 挂起
// - 调度器恢复执行
// - co_return 结束

// 4. 协程销毁
```

## Runtime 使用指南

### 基本设置

```cpp
#include "galay-kernel/kernel/Runtime.h"

using namespace galay::kernel;

int main() {
    // 方式 1：零配置（自动检测 CPU 核心数）
    Runtime runtime = RuntimeBuilder().build();
    runtime.start();

    // 方式 2：指定调度器数量
    Runtime runtime = RuntimeBuilder().ioSchedulerCount(4).computeSchedulerCount(2).build();  // 4 个 IO 调度器，2 个计算调度器
    runtime.start();

    // 获取调度器
    auto* io_scheduler = runtime.getNextIOScheduler();
    auto* compute_scheduler = runtime.getNextComputeScheduler();

    // 提交协程
    io_scheduler->spawn(myCoroutine());

    // 停止 Runtime
    runtime.stop();

    return 0;
}
```

## 网络编程指南

### TCP 服务器

```cpp
#include "galay-kernel/async/TcpSocket.h"
#include "galay-kernel/kernel/Runtime.h"

using namespace galay::kernel;
using namespace galay::async;

Coroutine handleClient(GHandle handle) {
    TcpSocket client(handle);
    client.option().handleNonBlock();

    char buffer[4096];
    while (true) {
        auto recv_result = co_await client.recv(buffer, sizeof(buffer));
        if (!recv_result || recv_result.value().size() == 0) {
            break;
        }

        auto& bytes = recv_result.value();
        std::cout << "收到: " << std::string(bytes.c_str(), bytes.size()) << '\n';

        // 回显
        auto send_result = co_await client.send(bytes.c_str(), bytes.size());
        if (!send_result) {
            break;
        }
    }

    co_await client.close();
}

Coroutine tcpServer(IOScheduler* scheduler) {
    TcpSocket listener;
    listener.option().handleReuseAddr();
    listener.option().handleNonBlock();

    auto bind_result = listener.bind(Host(IPType::IPV4, "0.0.0.0", 8080));
    if (!bind_result) {
        std::cerr << "绑定失败\n";
        co_return;
    }

    auto listen_result = listener.listen(1024);
    if (!listen_result) {
        std::cerr << "监听失败\n";
        co_return;
    }

    std::cout << "TCP 服务器监听 0.0.0.0:8080\n";

    while (true) {
        Host peer;
        auto accept_result = co_await listener.accept(&peer);
        if (!accept_result) {
            continue;
        }

        std::cout << "接受连接: " << peer.ip() << ":" << peer.port() << '\n';
        scheduler->spawn(handleClient(accept_result.value()));
    }
}

int main() {
    Runtime runtime = RuntimeBuilder().build();
    runtime.start();

    auto* scheduler = runtime.getNextIOScheduler();
    scheduler->spawn(tcpServer(scheduler));

    std::this_thread::sleep_for(std::chrono::hours(24));
    runtime.stop();

    return 0;
}
```

## 协程并发指南

### MpscChannel（多生产者单消费者）

```cpp
#include "galay-kernel/concurrency/MpscChannel.h"

Coroutine producer(MpscChannel<int>& channel, int id) {
    for (int i = 0; i < 10; ++i) {
        co_await channel.send(id * 100 + i);
        std::cout << "生产者 " << id << " 发送: " << (id * 100 + i) << '\n';
        co_await galay::kernel::sleep(std::chrono::milliseconds(100));
    }
}

Coroutine consumer(MpscChannel<int>& channel) {
    while (true) {
        auto value = co_await channel.recv();
        if (!value) {
            break;
        }
        std::cout << "消费者收到: " << *value << '\n';
    }
}

Coroutine channelExample(IOScheduler* scheduler) {
    MpscChannel<int> channel(scheduler, 10);  // 缓冲区大小 10

    // 启动多个生产者
    scheduler->spawn(producer(channel, 1));
    scheduler->spawn(producer(channel, 2));
    scheduler->spawn(producer(channel, 3));

    // 启动消费者
    scheduler->spawn(consumer(channel));

    co_await galay::kernel::sleep(std::chrono::seconds(5));

    // 关闭通道
    channel.close();
}
```

### AsyncMutex（异步互斥锁）

```cpp
#include "galay-kernel/concurrency/AsyncMutex.h"

AsyncMutex mutex;
int shared_counter = 0;

Coroutine incrementCounter(int id) {
    for (int i = 0; i < 10; ++i) {
        // 获取锁
        co_await mutex.lock();

        // 临界区
        int old_value = shared_counter;
        co_await galay::kernel::sleep(std::chrono::milliseconds(10));
        shared_counter = old_value + 1;
        std::cout << "协程 " << id << " 增加计数器: " << shared_counter << '\n';

        // 释放锁
        mutex.unlock();

        co_await galay::kernel::sleep(std::chrono::milliseconds(50));
    }
}

Coroutine mutexExample(IOScheduler* scheduler) {

    scheduler->spawn(incrementCounter(2));
    scheduler->spawn(incrementCounter(3));

    co_await galay::kernel::sleep(std::chrono::seconds(5));
    std::cout << "最终计数器值: " << shared_counter << '\n';
}
```

## 实战场景

### Echo 服务器（完整示例）

```cpp
#include "galay-kernel/kernel/Runtime.h"
#include "galay-kernel/async/TcpSocket.h"
#include <atomic>

using namespace galay::kernel;
using namespace galay::async;

std::atomic<int> connection_count{0};

Coroutine handleConnection(GHandle handle, int conn_id) {
    connection_count++;
    std::cout << "连接 #" << conn_id << " 建立，当前连接数: "
              << connection_count.load() << '\n';

    TcpSocket client(handle);
    client.option().handleNonBlock();
    client.option().handleTcpNoDelay();

    char buffer[4096];
    while (true) {
        auto recv_result = co_await client.recv(buffer, sizeof(buffer));
        if (!recv_result || recv_result.value().size() == 0) {
            break;
        }

        auto& bytes = recv_result.value();
        auto send_result = co_await client.send(bytes.c_str(), bytes.size());
        if (!send_result) {
            break;
        }
    }

    co_await client.close();

    connection_count--;
    std::cout << "连接 #" << conn_id << " 关闭，当前连接数: "
              << connection_count.load() << '\n';
}

Coroutine echoServer(IOScheduler* scheduler) {
    TcpSocket listener;
    listener.option().handleReuseAddr();
    listener.option().handleNonBlock();

    auto bind_result = listener.bind(Host(IPType::IPV4, "0.0.0.0", 8080));
    if (!bind_result) {
        std::cerr << "绑定失败\n";
        co_return;
    }

    auto listen_result = listener.listen(1024);
    if (!listen_result) {
        std::cerr << "监听失败\n";
        co_return;
    }

    std::cout << "Echo 服务器监听 0.0.0.0:8080\n";

    int conn_id = 0;
    while (true) {
        Host peer;
        auto accept_result = co_await listener.accept(&peer);
        if (!accept_result) {
            continue;
        }

        scheduler->spawn(handleConnection(accept_result.value(), ++conn_id));
    }
}

int main() {
    Runtime runtime = RuntimeBuilder().ioSchedulerCount(4).computeSchedulerCount(2).build();  // 4 个 IO 调度器，2 个计算调度器
    runtime.start();

    auto* scheduler = runtime.getNextIOScheduler();
    scheduler->spawn(echoServer(scheduler));

    std::cout << "按 Ctrl+C 停止服务器\n";
    std::this_thread::sleep_for(std::chrono::hours(24));

    runtime.stop();
    return 0;
}
```

Coroutine downloadFile(const std::string& url, int id) {
    TcpSocket socket;
    socket.option().handleNonBlock();

    // 解析 URL 并连接
    auto connect_result = co_await socket.connect(Host(IPType::IPV4, "example.com", 80));

    std::cout << "下载 " << id << " 完成，大小: " << response.size() << " 字节\n";

    co_await socket.close();

## 错误处理最佳实践

### 网络错误处理

```cpp
Coroutine robustNetworkOperation() {
    TcpSocket socket;
    socket.option().handleNonBlock();

    // 连接重试
    int max_retries = 3;
    for (int i = 0; i < max_retries; ++i) {
        auto connect_result = co_await socket.connect(Host(IPType::IPV4, "127.0.0.1", 8080));
        if (connect_result) {
            std::cout << "连接成功\n";
            break;
        }

        std::cerr << "连接失败，重试 " << (i + 1) << "/" << max_retries << '\n';
        if (i < max_retries - 1) {
            co_await galay::kernel::sleep(std::chrono::seconds(1));
        }
    }

    // 发送数据
    std::string message = "Hello";
    auto send_result = co_await socket.send(message.c_str(), message.size());
    if (!send_result) {
        std::cerr << "发送失败\n";
        co_await socket.close();
        co_return;
    }

    // 接收数据（带超时）
    try {
        auto result = co_await galay::kernel::timeout(
            std::chrono::seconds(5),
            [&]() -> Coroutine {
                char buffer[4096];
                auto recv_result = co_await socket.recv(buffer, sizeof(buffer));
                if (recv_result && recv_result.value().size() > 0) {
                    std::cout << "收到响应\n";
                }
                co_return;
            }()
        );
    } catch (const std::exception& e) {
        std::cerr << "接收超时\n";
    }

    co_await socket.close();
}
```

## 常见问题

### Q: 如何在协程中使用同步代码？

A: 使用 ComputeScheduler 执行同步代码：

```cpp
auto* compute_scheduler = runtime.getNextComputeScheduler();
compute_scheduler->spawn([&]() -> Coroutine {
    // 同步代码
    heavyComputation();
    co_return;
}());
```

### Q: 如何优雅地关闭服务器？

A: 使用信号处理和标志位：

```cpp
std::atomic<bool> running{true};

signal(SIGINT, [](int) {
    running.store(false);
});

while (running.load()) {
    // 服务器循环
}

runtime.stop();
```

### Q: 协程可以嵌套吗？

A: 可以，协程可以 co_await 其他协程：
